# applegame (Apple Box / Sum 10)

`fruit_box_a`(사과 게임/フルーツボックス) 스타일의 **드래그 선택 퍼즐 게임**입니다.

- 화면에는 **1~9 숫자를 가진 사과**가 **직사각형 그리드(가로 20 × 세로 10)** 로 배치됩니다.
- 마우스로 드래그해 **사각형 범위**를 만들고, 그 안에 들어온 사과들의 **숫자 합이 정확히 10**이면
  마우스를 놓는 순간 해당 사과들이 **사라집니다**.
- 제한 시간(기본 120초) 안에 **최대한 많은 사과를 없애서 점수(없앤 사과 개수)를 올리는** 것이 목표입니다.

## 플레이 방법

- **드래그**: 마우스(또는 터치)로 드래그해서 선택 박스를 만듭니다.
- **성공 조건**: 선택된 사과 숫자 합이 **10**이면 선택 박스가 성공 상태로 표시됩니다.
- **삭제**: 성공 상태에서 **마우스를 놓으면** 선택된 사과들이 삭제되고 점수가 올라갑니다.
- **리필 없음**: 사과가 사라진 칸은 **그대로 빈칸**으로 남습니다.

## 로컬에서 실행하기

프로젝트는 웹(Vite + TypeScript)로 만들어졌고, 실제 앱은 `web/` 폴더에 있습니다.

### 1) 설치

```bash
cd web
npm install
```

### 2) 개발 서버 실행

```bash
npm run dev
```

콘솔에 출력되는 주소(보통 `http://localhost:5173/`)로 접속하면 됩니다.

### 3) 프로덕션 빌드(선택)

```bash
cd web
npm run build
```

## 이미지(apple.png)

- 루트에 있는 `apple.png`를 사용합니다.
- 웹 앱에서는 빌드/실행 편의를 위해 `web/src/assets/apple.png`로 복사해 포함되어 있습니다.

## 폴더 구조

```text
applegame/
  apple.png
  web/
    index.html
    package.json
    src/
      main.ts              # UI + 게임 부팅
      style.css            # 화면 스타일
      game/
        createGame.ts      # 캔버스 게임 로직(그리드/드래그/합10/타이머)
      assets/
        apple.png
```

## 구현 메모

- 렌더링은 **Canvas 2D**를 사용합니다.
- 드래그 선택 박스는 **미리보기**가 되고, 포인터 이동은 프레임 단위로 **부드럽게 이어지도록(smoothing)** 처리되어 있습니다.
